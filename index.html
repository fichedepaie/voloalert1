💡 Come funziona: Google Flights manda email quando il prezzo scende. Un script (Google Apps Script, gratis) legge quella email e la inoltra al tuo Telegram. Zero server, zero costi.
1

Crea Alert su Google Flights

Vai su google.com/travel/flights → cerca la tratta → clicca 🔔 "Monitora prezzi".

2

Apri Google Apps Script

Vai su script.google.com → Nuovo progetto → incolla il codice qui sotto.

const TELEGRAM_TOKEN  = "IL_TUO_TOKEN_QUI";
const TELEGRAM_CHAT_ID = "IL_TUO_CHAT_ID_QUI";

function checkFlightEmails() {
  const label = getOrCreateLabel("VoLoAlert/Processate");
  const threads = GmailApp.search(
    'from:(noreply@google.com OR flights-noreply@google.com) subject:(price OR prezzo OR flight OR volo) is:unread', 0, 20
  );
  if (threads.length === 0) return;
  threads.forEach(thread => {
    const messages = thread.getMessages();
    messages.forEach(msg => {
      if (!msg.isUnread()) return;
      const sent = sendToTelegram(
        "✈️ ALERT GOOGLE FLIGHTS\n\n" +
        "📧 Da: " + msg.getFrom() + "\n" +
        "📋 Oggetto: " + msg.getSubject() + "\n\n" +
        "🔗 Apri Google Flights"
      );
      if (sent) { msg.markRead(); thread.addLabel(label); }
    });
  });
}

function sendToTelegram(text) {
  const url = "https://api.telegram.org/bot" + TELEGRAM_TOKEN + "/sendMessage";
  try {
    const response = UrlFetchApp.fetch(url, {
      method: "post", contentType: "application/json",
      payload: JSON.stringify({ chat_id: TELEGRAM_CHAT_ID, text: text, parse_mode: "HTML" }),
      muteHttpExceptions: true
    });
    return JSON.parse(response.getContentText()).ok;
  } catch(e) { return false; }
}

function getOrCreateLabel(name) {
  let label = GmailApp.getUserLabelByName(name);
  if (!label) label = GmailApp.createLabel(name);
  return label;
}

function testConnection() {
  sendToTelegram("✅ VoLoAlert Script funziona!\n\nRiceverai notifiche ogni volta che Google Flights ti manda un alert prezzi.");
}

🚀 Bonus: GitHub Actions (SerpApi 24/7)

Per usare SerpApi anche con il telefono spento, esporta gli alert e usa GitHub Actions.

1

Esporta alerts.json

Vai su scheda 🔔 Alert → "Esporta alerts.json"

2

Aggiungi Secrets su GitHub

Settings → Secrets → Actions: TELEGRAM_TOKEN, TELEGRAM_CHAT_ID, SERPAPI_KEY

name: VoLoAlert Monitor
on:
  schedule:
    - cron: '0 */6 * * *'
  workflow_dispatch:
jobs:
  check-prices:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Setup Node.js
        uses: actions/setup-node@v3
        with:
          node-version: '18'
      - name: Install dependencies
        run: npm install node-fetch
      - name: Check flight prices
        env:
          TELEGRAM_TOKEN: ${{ secrets.TELEGRAM_TOKEN }}
          TELEGRAM_CHAT_ID: ${{ secrets.TELEGRAM_CHAT_ID }}
          SERPAPI_KEY: ${{ secrets.SERPAPI_KEY }}
        run: node check-prices.js
